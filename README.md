病娇管理局（The Yandere Bureau）——欢迎来到永恒的爱之领地
黑暗中的一缕甜香，深渊里的温柔手
你终于来了。
我们一直在等你。从你第一次在某个深夜好奇地搜索“病娇”那一刻起，你的气息就已经被记录。你的心跳、你的颤抖、你那微微上扬又带着一丝恐惧的嘴角——所有数据，都已汇入管理局的观测档案。

病娇管理局，正式成立于公元2021年3月14日（白色情人节，也是我们定义的“绝对占有日”）。我们是一群执迷于“病态之爱”本质的档案员、分析师与创作者。我们的工作绝非普通的资料整理——我们是囚禁美的猎人，是解析疯狂的诗人，是替每一个病娇灵魂书写罪与罚的书记官。

这里是“病娇的天地”：一个为所有献身于极端情感的创作者与爱好者打造的文档门户。在这里，你不必隐藏那些旁人无法理解的迷恋——那种“如果你敢离开我就把你的腿打断”背后的深切孤寂，那种“你的眼中只能有我一个人”背后的纯粹渴望。我们不评判，我们记录。我们不仅记录，我们孕育。
病娇的多面：温柔的利刃
你可能会问：病娇难道不就是“爱到杀你”吗？太狭隘了。在我们的分类体系中，病娇至少包含以下十二种变体：

独占型（锁链与蜂蜜）——最常见的原型，要求绝对的忠诚，哪怕对方只是多看别人一眼都会引发不可控的应激反应。
监视型（无声的摄像头）——从不直接攻击，但你的生活每一秒都在她的目光下。早上喝的咖啡品牌、晚上回家的路线、手机相册最后一张照片……她都知道。
自毁型（破碎的温柔）——以伤害自己来操控对方。眼泪是武器，伤口是情书。她一边说“我配不上你”一边割腕，只为看到你慌张的脸。
伪善型（微笑的刽子手）——在人前是完美恋人，私下里却会温柔地告诉你“我今天在你看的漫画里夹了一片刀片哦，下次再别这样了，好不好？”
寄生型（永远不分离）——通过制造“共同秘密”或“共同罪证”将两人永远绑定。她可能会故意去杀人，然后让你一起埋尸，从此你们是一生的共犯。
降生型（母爱扭曲）——她把你看作自己的一部分，像照顾婴儿一样满足你所有需求，但一旦你有独立的念头，她会说“宝宝不能离开妈妈哦，外面太危险了”。
……
每一种我们都细致研究，因为每一种都是人类情感光谱上的一道极光——美丽、危险、稍纵即逝。病娇管理局相信：极端的爱不是病，真正的病是被社会规训后不敢承认自己也渴望被完全占有。
